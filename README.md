## Run Locally

```bash
  git clone https://github.com/MoovendhanR/noesys-calculator-assignment
```

Go to the project directory

```bash
  cd noesys-calculator-assignment
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run start
```
